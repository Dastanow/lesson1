Что такое HTML

HTML -HyperText Markup Language (язык гипертекстовой разметки )-представляет собой стандартизированный язык разметки документов и изпользуется для отображения веб-страниц в браузере. Это фармат разметки, с помощью которого браузер создает страницы сайтов при загрузке, формируя их из разных типов контента.

HTML (язык гипертекстовой разметки) не является языком программирования. Все что находится в HTML называются тегами.

HTML использует теги для идентификации различных типов контента и целей, которые каждый преследует на веб-странице

Какие бывают теги

  • Теги структуры HTML документа
  • Теги заголовков
  • Теги форматирования текста
